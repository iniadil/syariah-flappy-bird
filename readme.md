# Syariah Flappy Bird

A mobile-first Flappy Bird clone built with Phaser.js & Vite, featuring difficulty scaling and sound effects.

## Gameplay

- Tap or press **Space** to flap
- Fly through the gaps between pillars
- Every **5 points**, difficulty increases — pillars get faster and gaps get smaller
- Don't crash!

## Tech Stack

- [Phaser.js](https://phaser.io) 3 — game framework
- [Vite](https://vitejs.dev) — dev server & bundler

## Getting Started

```bash
git clone https://github.com/iniadi/syariah-flappy-bird
cd syariah-flappy-bird
npm install
npm run dev
```

Open your browser at `http://localhost:5173`

## Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder — host it on any static server.
